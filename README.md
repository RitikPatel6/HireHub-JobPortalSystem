# HireHub-JobPortalSystem

Job_Portal is a web-based recruitment platform where job seekers can search and apply for jobs, employers can post job vacancies, and admin can manage users, companies, and job posts.

## Features

- User registration and login
- Job seeker profile management
- Search and apply for jobs
- Employer can post and manage jobs
- Admin can manage users, companies, and job posts
- Application tracking system

## Technologies Used

- HTML
- CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MySQL / MongoDB

## Project Modules

### Job Seeker
- Create profile
- Upload resume
- Search jobs
- Apply for jobs
- Track application status

### Employer
- Post job vacancies
- View applications
- Shortlist candidates
- Manage job posts

### Admin
- Manage users
- Manage companies
- Manage jobs
- Monitor recruitment activities

## Conclusion

This Job Portal system helps to simplify the recruitment process by connecting job seekers, employers, and administrators on one centralized platform.
